# Source priority

When values disagree, use this priority order.

## Tier 1 - primary Renault / Renault Sport material

Use these as the strongest evidence.

- Renault Clio Renaultsport 182 Cup press release: official F4R 738 identity, bore, stroke, compression, rated power and torque.
- Renault F4R technical notes / workshop material: internal specifications, timing diagrams, phase shifter information, valvetrain information.
- Renault Sport Clio R3 assembly material: competition engine metrology such as chamber volume and minimum component weights.

## Tier 2 - OE-fit parts catalogues and specialist parts data

Useful for dimensions that are hard to find in public workshop text.

- Piston catalogues for compression height and rod/stroke compatibility.
- Valve catalogues for valve head diameter, stem diameter and length.
- Flywheel and clutch supplier data for stock mass comparisons.

These are supporting sources, not the primary authority for engine identity or official output.

## Tier 3 - respected Renault specialist and community material

Useful for:

- real dyno variance,
- common stock power curves,
- throttle body observations,
- manifold behaviour,
- known dephaser behaviour,
- practical measurements.

Community dynos should be treated as a band, not a single target.

## Tier 4 - simulator examples and community scripts

Used to understand Engine Simulator CE syntax and parameter behaviour.

Examples:

- Engine Simulator CE tutorials.
- AngeTheGreat example `.mr` files.
- Community Edition release notes.

These tell us how to represent a value, not what the Renault value is.

## Tier 5 - estimates

Only use estimates when no better source exists. Mark them clearly as estimates and replace them when measurement becomes available.
